# LDW (Laboratório de Desenvolvimento Web)
API do projeto Academia de musculação

## Tecnologias utilizadas:
 - Node.js
 - Express.js
 - Mongoose
 - MongoDB

## Começando
Simples assim  
Crie um arquivo **.env** na raíz do projeto, siga o arquivo **.env.example** na dúvida  
Depois:
```
npm i
npm start
```
